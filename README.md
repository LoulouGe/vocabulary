# Learn English

Une application web pour apprendre le vocabulaire anglais-français de manière ludique.

## Modes de jeu

### Quiz
Traduis les mots un par un (anglais vers français ou inversement). Un indice est donné en cas de mauvaise réponse. 10 mots par partie.

### Relier les mots
Associe 5 mots anglais à leur traduction française en cliquant sur les paires. Les bonnes réponses sont validées en vert, les erreurs clignotent en rouge.

## Thèmes disponibles

- La maison
- La nourriture
- Les animaux
- La nature
- L'école
- Les vêtements

Tu peux aussi créer ta propre liste de mots au format `anglais = français`.

## Lancer le projet

Ouvre `index.html` dans ton navigateur, ou lance un serveur local :

```bash
python3 -m http.server 8080
```

Puis ouvre `http://localhost:8080`.

## Technologies

HTML, CSS, JavaScript (vanilla) — aucune dépendance externe.
